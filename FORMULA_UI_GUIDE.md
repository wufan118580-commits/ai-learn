# 公式识别界面使用指南

## 界面布局说明

### 1. 上传和识别区域
```
┌─────────────────────────────────────────────────┐
│ 📤 选择图片                                    │
└─────────────────────────────────────────────────┘
```
点击按钮选择包含数学公式的图片（PNG、JPG、JPEG、BMP格式）

### 2. 识别结果验证区域
识别成功后，界面会分为左右两栏：

#### 左侧 - LaTeX代码编辑
```
┌──────────────────────────────────────┐
│ 📝 LaTeX代码 (可编辑)               │
│ ┌────────────────────────────────┐   │
│ │ E = mc^2                     │   │
│ │ [可以在这里直接编辑]          │   │
│ └────────────────────────────────┘   │
│                                    │
│ [🔄 重置为原始识别结果]             │
└──────────────────────────────────────┘
```

#### 右侧 - 实时预览
```
┌──────────────────────────────────────┐
│ 👀 公式预览                         │
│                                    │
│        E = mc²                      │
│     (渲染后的公式)                  │
│                                    │
│ 当前预览的LaTeX代码:                │
│ E = mc^2                           │
└──────────────────────────────────────┘
```

### 3. MathML代码区域（折叠）
```
┌──────────────────────────────────────┐
│ 🧮 查看MathML代码 (用于Word)       │
│ [点击展开/折叠]                     │

│ └────────────────────────────────┘   │
│                                    │
│ [📋 复制MathML代码]                 │
└──────────────────────────────────────┘
```

### 4. 导出文件区域
```
┌──────────────┬──────────────┬──────────────┐
│ 📥 下载      │ 📥 下载      │ 📥 下载      │

                  [📥 下载Markdown文件]
```

## 使用流程

### 完整工作流程
```
1. 上传图片
   ↓
2. 识别公式
   ↓
3. 查看预览（右侧）
   ↓
4. 判断是否正确
   ├─ 正确 → 直接导出
   └─ 错误 → 编辑修正
       ↓
5. 修改LaTeX代码（左侧）
   ↓
6. 实时查看预览效果（右侧）
   ↓
7. 确认无误后导出
```

### 典型使用场景

#### 场景1：识别正确
```
步骤1: 上传图片
步骤2: 点击"识别公式"
步骤3: 查看右侧预览 ✅ 公式正确
步骤4: 直接点击"下载LaTeX文件"或"下载Word文件"
```

#### 场景2：识别有误
```
步骤1: 上传图片
步骤2: 点击"识别公式"
步骤3: 查看右侧预览 ❌ 公式有误
步骤4: 在左侧编辑框修改LaTeX代码
步骤5: 右侧预览实时更新
步骤6: 重复步骤4-5直到满意
步骤7: 点击"下载LaTeX文件"或"下载Word文件"
```

#### 场景3：需要调整格式
```
步骤1: 上传图片
步骤2: 点击"识别公式"
步骤3: 查看右侧预览 ⚠️ 格式需要调整
步骤4: 在左侧编辑框修改（如：\frac{}{} 改为 \dfrac{}{}）
步骤5: 右侧预览实时更新
步骤6: 确认后导出
```

## 快捷键和操作

### 编辑快捷键
- `Ctrl + A` - 全选代码
- `Ctrl + C` - 复制代码
- `Ctrl + V` - 粘贴代码
- `Ctrl + Z` - 撤销（在编辑框内）

### 按钮说明
- `🔄 重置为原始识别结果` - 恢复到最初识别的LaTeX代码
- `📋 复制MathML代码` - 复制MathML代码到剪贴板
- `📥 下载` 各类文件 - 下载对应格式的文件

## 常见问题

### Q1: 如何判断识别是否正确？
A: 对比右侧预览和原始图片：
- 公式符号是否一致
- 公式结构是否正确
- 下标、上标位置是否准确
- 特殊符号（如积分、求和）是否正确

### Q2: 识别错误如何修正？
A: 在左侧编辑框直接修改：
- 查看LaTeX语法规则
- 参考类似公式的LaTeX写法
- 实时查看右侧预览效果
- 不满意可以随时重置

### Q3: 预览显示乱码或错误？
A: 检查以下几点：
- LaTeX代码语法是否正确
- 特殊符号是否转义（如 `\`）
- 括号是否匹配
- 右侧会显示具体错误信息

### Q4: 如何在Word中使用？
A: 方法：
1. 点击"🧮 查看MathML代码"展开区域
2. 点击"📋 复制MathML代码"
3. 在Word中按 Ctrl+V 粘贴
4. 公式自动转换为Word格式

### Q5: 修改后的代码会影响历史记录吗？
A: 不会。历史记录保存的是原始识别结果。修改只影响当前会话的导出。

## 高级技巧

### 1. 批量处理
如果有多张类似的公式图片：
- 识别第一张，修正模板
- 后续图片可以参考模板格式
- 在左侧编辑框快速调整

### 2. 符号替换
常见的识别错误：
- `\alpha` → `\a` (修正为 `\alpha`)
- `\beta` → `\b` (修正为 `\beta`)
- `\sum` → `\Sigma` (修正为 `\sum`)

### 3. 格式调整
美化公式格式：
- `\frac{a}{b}` → `\dfrac{a}{b}` (更大分式)
- `\int` → `\displaystyle\int` (更大积分号)
- 使用 `\,` 或 `\;` 调整间距

### 4. 预览技巧
- 如果公式太长，可以分段预览
- 使用 `\quad` 增加间距让公式更清晰
- 复杂公式可以先简化预览，确认结构后再完善

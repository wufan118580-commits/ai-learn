### 1. 新增可视化预览和编辑功能 ✨

#### 功能描述
- 左右分栏布局：左侧可编辑LaTeX代码，右侧实时预览公式

#### 界面改进
```
┌──────────────────┬──────────────────┐
│  📝 LaTeX代码    │   👀 公式预览    │
│  (可编辑)        │   (实时更新)     │
│                  │                  │
│  [编辑框]        │  [渲染结果]      │
│                  │                  │
│  [🔄 重置]       │                  │
└──────────────────┴──────────────────┘
```

#### 使用方法
1. 识别公式后，查看右侧预览效果
2. 如果识别有误，在左侧编辑框修改LaTeX代码
3. 右侧预览会实时更新，显示修改后的效果
4. 确认无误后，点击下载按钮导出

## 技术细节

### 实现方式
- 使用Streamlit的`st.text_area`实现可编辑文本框
- 使用`st.markdown`和LaTeX语法实现实时预览
- 通过session state管理编辑状态
- 使用`st.rerun()`实现界面刷新

### 关键代码
```python
# 可编辑的LaTeX文本框
edited_latex = st.text_area(
    "编辑LaTeX代码",
    value=st.session_state.editable_latex,
    height=200,
    key="editable_latex_text"
)

# 实时预览
preview_latex = f"$${edited_latex}$$"
st.markdown(preview_latex, unsafe_allow_html=True)
```
